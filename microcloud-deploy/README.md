# Get started

```
sudo lxd init --auto
```

## Init tofu

```
sudo snap install --classic tofu
tofu init
```

## Apply the plan

```
tofu apply
```

## Init Microcloud

For now microcloud initialisation needs to happen by hand. To do this follow the docs: https://canonical-microcloud.readthedocs-hosted.com/en/latest/how-to/initialise/
